## 使用样例

#### Ethereum

- **f2pool:** HSPMiner.exe -epool eth.f2pool.com:8008 -ewal {替换钱包地址} -eworker {替换旷工名称} -epsw {替换密码} -logfile -api 0.0.0.0:16666
